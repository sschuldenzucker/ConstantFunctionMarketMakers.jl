
# ConstantFunctionMarketMakers.jl

Library for constant function market makers, as used in blockchains. See the [readme](https://github.com/sschuldenzucker/ConstantFunctionMarketMakers.jl) for quickstart and overview and [API Reference](api.md) for details.

To define new AMMs and for other special applications, see the [Internal API](api_internal.md).
